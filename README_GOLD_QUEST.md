# Gold Quest (Beginner)

Run the simple terminal game to reach $10,000.

Run:
```bash
python gold_quest.py
```

Commands inside the game:
- `collect` or `c`: collect money
- `buy` or `b`: buy multiplier upgrade (cost increases)
- `disadvantage` or `d`: take instant cash but increases penalties
- `remove` or `r`: remove one disadvantage for a cost
- `status` or `s`: show current stats
- `help` or `h`: show help
- `quit` or `q`: exit

Enjoy! This is intentionally simple for beginners — modify as you like.
